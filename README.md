# Cameron Lu Website

Elegant static website for Cameron Lu, a Sydney pianist, chamber music performer, and teacher.

## Preview locally

Open `index.html` through a local static server, for example:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173`.

## Publish free

This is a static website, so it can be published on a free starter URL with:

- Cloudflare Pages: `cameron-lu.pages.dev`
- Netlify: `cameron-lu.netlify.app`
- Vercel: `cameron-lu.vercel.app`
- GitHub Pages: `username.github.io/cameron-lu`

The site does not require WordPress unless you want a CMS dashboard for editing content.
